# 웹 개발이란

포트폴리오 사이트나 기업 홍보사이트처럼 방문자에게 정보를 보여주는 웹 사이트는 간단하다. 이런 사이트를 정적 사이트, static site 라고 한다.

그러나 웹 개발은 보여주는게 전부가 아니다. 예를 들면 페이스북, 유튜브 같이 여러가지 동적인 기능을 구현해야 하는 경우가 많다. 이러한 사이트를 dynamic site라고 하고, 따라서 웹개발은 사용자들에게 제공할 기능과 서비스까지 모두 준비해야 한다. 

# 서버와 클라이언트 

사용자가 웹사이트에 접근하려고 사용하는 pc, tablet, phone 등을 클라이언트 라고 한다. 좁은 의미로는 웹브라우저를 클라이언트라고 한다. 
클라이언트로 접근해서 정보를 가져와서 보여줄 수 있도록 제공하는 컴퓨터를 서버라고 한다. 

# 프론트엔드. 백엔드 

1. FRONTEND
사용자 앞에 보이는 영역을 프론트엔드, 즉 브라우저의 화면에 보이는 것들을 다룬다. 
 - HTML, CSS, JS

2. BACKEND
사용자의 뒤에서 보이지 않는 영역을 뜻한다. 회원정보, 게시물, 사진들 데이터 베이스에 저장된다. 데이터베이스를 설계하거나 데이터를 처리하는 것이 백엔드 개발 영역이다. 
JAVA, PHP, ASP, PYTHON(JANGO, PLASK), JS
